使用Hexo+Github Page搭建博客

遇到的坑：

  1. 在使用hexo d 推送新的内容时，绑定的域名有时会掉线，找了好久发现是hexo在部署的时候自动删除CNAME文件,解决方式是将 image ,favicon.io CNAME 在本地推送时放入 source 目录，这样就可以避免文件被删除。具体操作是将推送到github的文件克隆到本地，然后将需要的文件找出来放入source 目录。

注意：CNAME文件存放的就是域名，比如我的CNAME文件内容，只有一句"lichengxiang0.top"

  1. 在搭建环境将内容推送到Github时，修改的内容如下：
# Deployment
## Docs: https://hexo.io/docs/deployment.html
deploy:
  type: git
  github: https://github.com/lichengxiang0/lichengxiang0.github.io
  branch: master

需要注意格式，github+英文冒号+空格+地址
